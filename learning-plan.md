> 本计划会根据每日学习进度和课程安排动态调整。

## 当前阶段

**阶段一：建立共同语言** —— 按 Handbook 大纲补全 AI × Web3 交叉领域的概念地图。

## 学习路线

### AI 基础复习（Week 1-2）

按 Handbook AI 基础章节快速过一遍，重点补足：
- [ ] [Agent](https://aiweb3.school/zh/handbook/ai/agent/) — Agent 工具调用与多步执行
- [ ] [MCP](https://aiweb3.school/zh/handbook/ai/mcp/) — 模型、工具、上下文的协议连接
- [ ] [Frameworks](https://aiweb3.school/zh/handbook/ai/frameworks/) — LangChain / LangGraph / Agents SDK 的编排层

### Web3 基础复习（Week 2-3）

- [ ] [Smart Contract](https://aiweb3.school/zh/handbook/web3/smart-contract/) — 链上规则调用
- [ ] [Account Abstraction](https://aiweb3.school/zh/handbook/web3/account-abstraction/) — Smart Account 与 Agent 权限
- [ ] [Wallet](https://aiweb3.school/zh/handbook/web3/wallet/) — 身份与签名入口

### AI × Web3 Bridge（Week 3-5）

核心交叉领域，重点攻坚：
- [ ] [Web3 Tool Use](https://aiweb3.school/zh/handbook/bridge/web3-tool-use/) — Agent 调用 RPC、钱包、合约
- [ ] [Agent Wallet](https://aiweb3.school/zh/handbook/bridge/agent-wallet/) — Agent 权限、限制与撤销
- [ ] [Machine Payment](https://aiweb3.school/zh/handbook/bridge/machine-payment/) — 机器间小额支付
- [ ] [Agent Identity](https://aiweb3.school/zh/handbook/bridge/agent-identity/) — Agent 识别与授权
- [ ] [AI Security](https://aiweb3.school/zh/handbook/bridge/ai-security/) — Prompt Injection、权限隔离

### 前沿探索 / Hackathon 准备（Week 5+）

- [ ] [智能体商业](https://aiweb3.school/zh/handbook/tracks/agentic-commerce/)
- [ ] [钱包与权限](https://aiweb3.school/zh/handbook/tracks/wallet-permission/)
- [ ] 根据 Cohort 任务和 Hackathon 主题确定具体项目方向
