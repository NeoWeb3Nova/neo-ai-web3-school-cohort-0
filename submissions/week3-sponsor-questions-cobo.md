# Week 3｜赞助方问题收集｜给 Cobo / Z.AI 的问题

## 问题一

**赞助方：** Cobo

**问题：** AI Agent 自主管理链上资产时，如何通过 MPC 或智能合约钱包实现既保证 Agent 的自主性，又防止私钥泄露或恶意交易？

**背景 / 卡点：**
在构建 AI × Web3 项目时，AI Agent 需要能够自主执行链上操作（如交易、质押、跨链交互），但传统 EOA 钱包将私钥暴露给 Agent 存在巨大安全风险。目前卡在：如何在不暴露完整私钥的前提下，让 AI Agent 安全地控制资产并执行自动化策略。

**希望分享会回答什么：**
1. Cobo 的 MPC 钱包或智能合约钱包方案中，是否有针对 AI Agent 场景的特殊设计？
2. 具体的集成架构是怎样的（例如 Agent 决策与签名授权的分离机制）？
3. 对于小额高频的 AI 自动化交易 vs 大额低频的资产管理，分别推荐哪种托管方案？
