## 笔记与思考

### 关键概念

1. **钱包创建三件套**
   - 助记词（Mnemonic）→ 派生多组私钥 → 公钥 → 地址
   - 安全原则：助记词/私钥永不进入 Agent 上下文或 public repo
   - 测试网专用：创建独立测试钱包，与主网资产隔离

2. **测试网交互链路**
   - 选择测试网（Sepolia / Holesky / L2 测试网）→ 配置 RPC → 领水（Faucet）→ 发送交易 → 浏览器验证
   - Faucet 可能需要主网小额资产或社交验证（如 Alchemy / Infura / 官方 Faucet）

3. **合约部署最小闭环**
   - Remix：写合约 → Compile → Inject Web3 / Environment → Deploy → 获取合约地址
   - 读取调用（view/pure）：不消耗 Gas，直接返回状态
   - 写入调用（transaction）：需签名、消耗 Gas、上链确认
