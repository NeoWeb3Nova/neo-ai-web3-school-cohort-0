## 笔记与思考

### 关键概念

1. **打卡 ≠ 完成任务**
   官方明确两者并行。我需要在每日笔记中同时记录：任务完成证明 + 打卡草稿，缺一不可。

2. **学分 = 长期资产**
   评优、周边奖励、后续资源推荐都挂钩。每天的质量不能敷衍。

3. **模块 B 核心链路**
   助记词 → 私钥 → 地址（钱包）→ 签名（授权动作）→ 交易（Gas 付费）→ 链上确认 → 浏览器验证

4. **安全底线**
   - 地址 ≠ 匿名
   - 签名 ≠ 普通登录
   - 授权 ≠ 转账
   - AI Agent 不应直接接触私钥 / 助记词
   - 涉及签名、授权、转账、合约写入必须保留人工确认
