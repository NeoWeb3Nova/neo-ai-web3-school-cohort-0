│           ↑                                   ↑                    │

## 流程层次

```
┌──────────────────────────────────────────────────────────────────┐
│  ┌────────────────────────────────────────────────┐ │
│  │           场景层（Scenario）          │ │
│  │  Agent 购买的是什么？→ AI 推理服务、内容生成  │ │
│  └────────────────────────────────────────────────┘ │
│                          ↓                        │
│  ┌────────────────────────────────────────────────┐ │
│  │            流程层（Flow）              │ │
│  │  发现 → 报价 → 授权 → 执行 → 交付 → 验收 → 结算  │ │
│  └────────────────────────────────────────────────┘ │
│                          ↓                        │
│  ┌────────────────────────────────────────────────┐ │
│  │          验证层（Verification）          │ │
│  │  自动验证：交付物 hash 、违禁词扫描      │ │
│  │  人工验收：调性、创意、品牌匹配       │ │
│  │  仲裁机制：Evaluator 或超时自动退款      │ │
│  └────────────────────────────────────────────────┘ │
│                          ↓                        │
│  ┌────────────────────────────────────────────────┐ │
│  │           协议层（Protocol）            │ │
│  │  x402：支付通道 （HTTP 402 握手）          │ │
│  │  ERC-8183：Escrow + Evaluator            │ │
│  │  ERC-8004：身份 + 声誉                  │ │
│  │  CAW Pact：预算 + 权限 + 审计         │ │
│  └────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────┘
```

## 数据流

```
[商户 Alice]
      │  1. 确认预算 + 需求
      ↓
[CAW Wallet]
      │  2. 创建 Pact（金额/范围/时间）
      ↓
[Buyer Agent]
      │  3. 发现服务、见证声誉
      │  4. 发起 HTTP 请求
      ↓
[x402 Paywall Server]
      │  5. 返回 402 + 支付要求
      ↓
[Buyer Agent]
      │  6. 识别 402
      │  7. 查询 CAW Pact 预算
      │  8. 在授权内签名支付
      │  9. 重试带 Payment Proof
      ↓
[x402 Paywall Server]
      │  10. Facilitator 验证支付
      │  11. 执行 AI 推理
      │  12. 返回结果 + Receipt
      ↓
[Buyer Agent]
      │  13. 接收结果
      │  14. 提交交付物给 Alice 验收
      ↓
[商户 Alice]
      │  15. 人工验收 / 否决 / 仲裁
```

## 状态机

```
                    ┌────────────┐
                    │   开始   │
                    └────────────┘
                           │
            ┌───────────────────────────────────┐
            │              ↓                  │
            │   ┌───────────────────────┐   │
            │   │  预算已授权（Pact Active） │   │
            │   └───────────────────────┘   │
            │              │                  │
       ┌─────←────←─────────────────────────────┘
       │                        ↓
       │      ┌─────────────────────────────────────┐
       │      │        等待支付证明（Awaiting Payment）         │
       │      └─────────────────────────────────────┘
       │                        │
  拒绝/超时                    │ 支付成功
       │                        ↓
       │      ┌─────────────────────────────────────┐
       │      │           执行服务（Service Executing）          │
       │      └─────────────────────────────────────┘
       │                        │
       │                  交付完成
       │                        ↓
       │      ┌─────────────────────────────────────┐
       │      │          等待验收（Pending Acceptance）          │
       │      └─────────────────────────────────────┘
       │          │              │              │
   超时退款      │←←←←←─←←←←←←─────→
       │     │←退款        验收通过→结算
       │                  │              │
       │                  │              ↓
       │                  │    ┌──────────────┐
       │                  │    │ 完成（Completed） │
       │                  │    └──────────────┘
       │                  ↓
       │    ┌─────────────────────────────────────┐
       →→→│  争议仲裁（Dispute Arbitration）          │
            └─────────────────────────────────────┘
            │              │              │
         仲裁通过→结算    →→→  仲裁失败→退款
```
